# snake
Legendary game.

<h2>Demo</h2>
An online demo is available <a href="https://megasnake.herokuapp.com/">here</a>.

<h2>Production</h2>
To start the application in the production server run:<br>
<code>yarn start</code>

<ul>

  <li>express.js</li>
  <li>socket.io</li>
</ul>

The application frontend uses:

<ul>
  <li>html, css</li>
  <li>pixiJS</li>
</ul>
